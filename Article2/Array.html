Apa itu array ?

Array adalah sebuah kumpulan data yang terdiri dari elemen-elemen yang memiliki nilai (value).

Di banyak bahasa pemrograman, elemen-elemen di dalam array mempunyai jenis / tipe data yang sama, sehingga kita akan mendapati array of integer, array of string, dll. Setiap elemen di dalam array menempati sebuah lokasi yang diberikan kode angka sebagai alamat untuk membedakan antara satu lokasi dengan lokasi yang lain. Kode alamat ini disebut dengan index.

Di mayoritas bahasa pemrograman, seperti Java, PHP, Javascript, C, index dari suatu array diawali dari angka 0 (nol). Oleh karena itu, sebuah array yang terdiri dari 5 elemen, akan mempunyai index 0 hingga 4. Apa yang dimaksud dengan Array Multi Dimensi ?

Array yang kita bahas sebelumnya hanya memiliki satu index dan bisa juga kita sebut array satu dimensi.
Mungkin akan muncul pertanyaan, kalau ada array 1 dimensi, apakah ada array 2 dimensi atau bahkan 3 dimensi? Jawabannya adalah ADA. Banyak bahasa pemrograman yang menyediakan fitur, untuk menggunakan array dengan dimensi lebih dari satu atau juga disebut array multi dimensi.

Bagaimana dengan array 4 dimensi, 5 dimensi dan seterusnya?
Kita dapat melakukan cara yang sama seperti array 2 dimensi maupun 3 dimensi yang baru saja kita bahas. Pada array 3 dimensi, kita menggunakan 3 kurung siku untuk mendeklarasikan array tersebut. Pada array 4 dimensi, kita menggunakan 4 kurung siku. Pada array 5 dimensi kita menggunakan 5 kurung siku, dan seterusnya.

Jika kita perhatikan kembali array 3 dimensi kita, elemen di dimensi pertama merupakan sebuah array, yang terdiri dari 3 elemen dan masing – masing elemen tersebut merupakan sebuah array yang terdiri dari 5 elemen. Hal yang sama berlaku juga pada array 4 dimensi, 5 dimensi, dan seterusnya. Karena sesungguhnya, array multidimensi adalah suatu array di dalam array.

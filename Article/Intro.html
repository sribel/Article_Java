Koding Java untuk Pemula
1.Intro: Pengenalan

Halo teman-teman

Pada kesempatan ini, kita ingin membagikan tutorial untuk belajar koding bahasa pemrograman Java untuk pemula. Bagi teman-teman yang sebelumnya belum pernah koding, tapi sudah mengenal komputer, maka tutorial ini cocok untuk digunakan oleh teman-teman yang ingin belajar koding Untuk teman-teman yang masih awam di dunia koding, maka ikutilah tutorial ini secara urut sehingga mudah untuk dipahami.

Sedangkan bagi teman-teman yang sudah pernah koding dan ingin mempelajari beberapa topik tertentu, maka bisa langsung menuju ke tutorial yang diinginkan.

Apa itu Koding?

Mungkin di antara teman-teman ada yang bertanya, apa itu koding? Sebelumnya, mari kita lihat di kehidupan kita sehari-hari. Jika kita memberikan perintah ke seseorang, kita tentu akan menggunakan bahasa yang dipahami oleh orang tersebut, misalkan bahasa Indonesia.

Nah hal yang sama berlaku juga untuk komputer. Jika kita ingin memberikan perintah agar komputer melakukan sesuatu yang kita inginkan, kita juga harus memberikan perintah menggunakan bahasa yang dimengerti oleh komputer. Bahasa inilah yang disebut bahasa pemrograman.

Nah Koding sendiri adalah suatu proses di mana kita mengetikkan instruksi atau perintah untuk dijalankan oleh komputer menggunakan sebuah bahasa pemrograman

Ada banyak sekali bahasa pemrograman yang ada dan tentunya tidak mungkin kita mempelajari semuanya. Sesungguhnya, jika kita sudah menguasai konsep dasar pemrograman dan salah satu bahasa pemrograman yang ada, kita akan dengan mudah mempelajari bahasa pemrograman yang lain.

Nah, tutorial ini akan mengajarkan konsep dasar pemrograman untuk pemula dan untuk bahasa pemrograman yang akan kita pelajari, kita akan menggunakan bahasa pemrograman Java.

Instalasi NetBeans

Kita bisa menggunakan text editor seperti NotePad untuk membuat program / koding di bahasa pemrograman Java. Akan tetapi, untuk kemudahan dan kenyamanan, kita seharusnya menggunakan alat yang tepat yaitu sebuah IDE (Integrated Development Environment).

Salah satu IDE yang sangat populer untuk digunakan dalam pemrograman Java adalah NetBeans. Oleh karena itu, mari kita melakukan instalasi NetBeans di komputer kita untuk mengikuti tutorial ini.

Download dan Install JDK

  1. Pertama-tama kita buka aplikasi web browser, dalam hal ini kita memakai google chrome, lalu kita masuk ke situs resmi Oracle yaitu (www.oracle.com/java/technologies/downloads)
  2. Di sini kita bisa memilih berbagai versi JDK yang ingin kita download, akan tetapi disarankan untuk mendownload JDK versi terbaru, dalam hal ini kita akan mendownload JDK Versi 23.
  3. Sebelum mendownload JDK, kita diminta untuk memilih versi JDK yang sesuai dengan sistem operasi kita masing – masing. Untuk tutorial kali ini kita akan menggunakan versi JDK untuk Windows.
  4. Selanjutnya klik link untuk men-download installer dan tunggu hingga proses download selesai.
  5. Setelah proses download selesai, kita bisa minimize google chrome kita, lalu kita buka file exploler, selanjutnya menuju ke folder download. Di folder download ini terdapat file installer JDK Versi 23 yang telah kita download, dan langsung saja kita double klik file installer tersebut.
  6. Bila Sudah muncul wizard installasi seperti ini, klik next. Selanjutnya akan diberikan pilihan di mana kita akan menginstall JDK tersebut. Jika tidak ada preferensi khusus biarkan default, dan klik next. Selanjutnya tunggu installasi JDK hingga selesai. Bila sudah selesai, klik tombol close.
  7. Selanjutnya kita akan melakukan verifikasi JDK yang telah kita install dengan cara, buka Command Prompt, ketik ‘java -version’ lalu tekan Enter. Jika instalasi berhasil, kita akan melihat versi JDK yang telah terpasang.

Setelah memastikan bahkan JDK telah terpasang di computer maupun laptop kita masing – masing, langkah selanjutnya adalah men-download dan menginstall Apache Netbeans sebagai IDE kita selama melakukan koding Java :

Download dan Install Apache Netbeans

  1. Kita Kembali ke google chrome, tambahkan tab baru dengan cara klik tombol "+" berikut ini. Lalu kita masuk ke situs resmi apache netbeans yaitu netbeans.apache.org/front/main/download
  2. Di sini kita bisa memilih berbagai versi Apache Netbeans yang ingin kita download, akan tetapi disarankan untuk mendownload Apache Netbeans versi terbaru dalam hal ini kita akan mendownload Apache Netbeans Versi 23. Klik tombol download berikut
  3. Pada halaman download ini kita diminta untuk memilih veri Apache Netbeans yang sesuai dengan sistem operasi kita masing – masing. Untuk tutorial kali ini kita akan menggunakan versi Apache Netbeans untuk Windows. Selanjutnya klik link di bawah kata kata “we suggest the following location for you download” berikut ini. Tunggu hingga proses download selesai
  4. Setelah proses download selesai, kita bisa minimize google chrome kita kembali, lalu kita buka file exploler kita lagi, selanjutnya menuju ke folder download . Di folder download ini terdapat file installer Apache Netbeans Versi 23 yang telah kita download, dan langsung saja kita double klik file installer tersebut. Tunggu hingga configuration installer selesai dan nantinya akan muncul wizard installasi seperti berikut. Lalu klik next,
  5. Kemudian akan muncul lisensi agreement seperti berikut, dan kemudian check saja lisensi agreement di bawah ini. Lalu klik next lagi.
  6. Selanjutnya bisa kita lihat bahwa apache netbeans tersebut akan diinstall di direktori c/program files/ netbeans-23. Dan di bawahnya ini merupakan lokasi di mana folder jdk yang telah kita install sebelumnya. Bila tidak ada preferensi khusus di mana kita harus menginstall apache netbeans tersebut, biarkan default, lalu klik next.
  7. Selanjutnya langsung saja kita install dengan klik tombol install berikut ini. Tunggu hingga proses instalasi selesai.
  8. Setelah instalasi selesai, klik finish.

First Java Program

Setelah kita selesai melakukan installasi dan konfigurasi NetBeans di komputer kita, mari kita buat program Java pertama kita dengan langkah-langkah berikut ini

  1. Buatlah project baru:
    • Dengan cara klik New Project
    • Pilih Categories Java With Ant, dan pilih Projects: Java Class Library. Lalu klik next.
    • Isilah nama project misal : tutorial, dan lokasi penyimpanan project misal di d:\java. Lalu klik finish.
  2. Selanjutnya kita Buat file baru:
    • Dengan cara klik New File
    • Pilih Categories Java dan pilih File Types: Java Main Class
    • Isi class name dengan nama class misalkan Hello, lalu klik finish.
    • Untuk memperbesar tampilan kita bisa gunakan Alt+Scroll, menggunakan touchpad ataupun mouse kita masing masing.

    Penjelasan Program

    public class Hello {
        public static void main (String [] Args){

        } 
}

    Kemudian, di bawah public static void main, di antara kurung kurawal, ketiklah instruksi berikut ini:

    System.out.println("Hello World");

    Tanda titik kita lafalkan sebagai dot.

    public class Hello {
          public static void main (String [] Args){
            System.out.println("Hello World");
          } 
  }

    Perhatikan pengetikan huruf besar dan huruf kecil, dikarenakan di mayoritas bahasa pemrograman, huruf besar dan huruf kecil dianggap berbeda. Dan jangan lupa untuk mengetikkan tanda titik koma di akhir instruksi.

    Mari kita coba untuk mengeksekusi program tersebut dengan memilih menu Run -> Run File atau bisa juga langsung menekan tombol Shift-F6Perhatikan hasil dari eksekusi program kita di Output Windows. Kita mencetak kalimat Hello World di layar.

    Mari kita lihat kembali program pertama yang telah kita buat

     public class Hello  {
          public static void main (String [] Args){
            System.out.println("Hello World");
          } 
  }

    Di baris pertama ada instruksi public class Hello. Instruksi ini adalah deklarasi sebuah class yang bernama Hello. Perhatikan bahwa nama class sama dengan nama file kita. Dan aturan di Java, nama class harus diawali dengan huruf besar. Setelah itu, terdapat kurung kurawal buka dan kurung kurawal tutup sebagai penanda batas dari kode implementasi class tersebut. 

    public class Hello {
       public static void main (String [] Args) {
          System.out.println("Hello World");
        } 
}

    Di dalam kurung kurawal, terdapat instruksi public static void main(String[ ] args). Instruksi ini adalah suatu deklarasi function (fungsi) bernama main (untuk selanjutnya kita sebut main function). Untuk sementara ini, cukup kita ingat bagaimana kita mendeklarasikan main function ini. Pembahasan lebih detail tentang function akan dilakukan di tutorial selanjutnya. Oleh karena itu, ikutilah terus tutorial ini. 

    public class Hello {
        public static void main (String [] Args){
          System.out.println("Hello World");
          {
            System.out.println("Ini adalah program java pertama saya");
            System.out.println("Saya belajar koding di Ozeva Coding Academy");
          }  
       } 
}

    Setelah deklarasi function, kita juga gunakan kurung kurawal buka dan tutup yang berfungsi sebagai batas dari implementasi suatu function. Nah kita mengetikkan kode kita di antara kurung kurawal buka dan tutup ini. 

    public class Hello {
        public static void main (String [] Args){
          System.out.println("Hello World");
          {
            System.out.println("Ini adalah program java pertama saya");
            System.out.println("Saya belajar koding di Ozeva Coding Academy");
          }
        } 
}

    Instruksi System.out.println(“Hello World”); adalah suatu instruksi untuk mencetak ke layar apa yang kita tuliskan di antara tanda kutip, yang di instruksi ini adalah Hello World. Lihatlah di bagian Output yang juga tercetak Hello World.

    Di dalam istilah koding / pemrograman, instruksi System.out.println ini disebut statement dan di bahasa pemrograman Java, statement diakhiri dengan tanda ;. Selain itu, kita juga melihat adanya kurung kurawal buka dan tutup sebagai pembatas implementasi class dan juga fungsi.

    public class Hello {
        public static void main (String [] Args){
          System.out.println("Hello World");
        {
            System.out.println("Ini adalah program java pertama saya");
            System.out.println("Saya belajar koding di Ozeva Coding Academy");
          }  
        } 
}

    Di bahasa pemrograman Java, kita juga bisa mengelompokkan beberapa statement yang kita tuliskan di antara kurung kurawal buka dan tutup. Statement-statement ini disebut block of statements atau disingkat block saja seperti contoh diatas

    public class Hello {
            public static void main (String [] Args){
              System.out.println("Hello World");
             {
                System.out.println("Ini adalah program java pertama saya");
                System.out.println("Saya belajar koding di Ozeva Coding Academy");
              }  
            } 
    }

    Konsep block ini akan sangat penting karena statement-statement di dalam block ini akan dianggap sebagai satu kesatuan dan akan dieksekusi secara keseluruhan secara berurutan. Selain itu, masih ada beberapa aturan yang berhubungan dengan block ini yang akan kita bahas di tutorial selanjutnya.

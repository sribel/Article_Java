Pada kesempatan ini kita akan membahas tentang Variable

Apa itu variable?

Dalam coding atau pemrograman, variable adalah suatu tempat yang kita gunakan untuk menyimpan suatu data atau nilai. Dalam pemrograman, variable terdiri dari 3 komponen yaitu:

  • Nama
  • Tipe Data
  • Nilai/Value

Di bahasa pemrograman Java, untuk menggunakan variable ini, kita harus membuat variable tersebut terlebih dahulu. Proses pembuatan variable disebut deklarasi variable (variable declaration).

Di bahasa pemrograman Java, pada saat mendeklarasikan variable kita harus menentukan nama dan tipe dari variable tersebut.

Contoh:

 int var1;

Kita mendeklarasikan variable dengan nama var1 dengan tipe int. int singkatan dari integer yang berarti bilangan bulat. Setelah deklarasi, kita bisa menentukan nilai dari variable var1 dengan cara: 

 var1=5;   //var1 sama dengan 5

Menentukan nilai dari suatu variable disebut variable assignment dan menggunakan tanda sama dengan (=).

Kita juga bisa langsung mendeklarasikan variable dan langsung menentukan nilai dari variable tersebut dengan cara:

 int var1=5;  //integer var1 sama dengan 5

Nama Variable

Kita menggunakan nama var1 untuk variable pertama kita. Nah mungkin akan muncul pertanyaan, apakah kita bisa memberikan nama variable sesuka hati kita?

Kita bisa menggunakan nama variable apapun asalkan tidak melanggar beberapa aturan berikut:

  • Nama variable hanya boleh terdiri dari huruf (baik huruf besar maupun huruf kecil), angka, tanda $ (dolar) dan tanda _ (underscore/garis bawah).
  • Nama variable tidak boleh diawali dengan angka.
  • Nama variable tidak boleh sama dengan keywords. Apa itu keywords? Keywords adalah kata-kata yang digunakan di dalam bahasa pemrograman dan sudah memiliki fungsi atau konteksnya sendiri, seperti int, double, if, else, for, dll. Jadi kita tidak boleh menggunakan int, double, if, else, for, dll untuk nama variable kita.

Selain ketiga aturan wajib di atas, di komunitas programmer Java juga muncul beberapa ketentuan yang diikuti oleh mayoritas programmer Java untuk pemberian nama variable ataupun nama lain yang disebut Naming Convention. Naming Convention di Java antara lain:

  • Nama variable diawali dengan huruf kecil.
  • Nama variable yang terdiri dari dua kata atau lebih ditulis menggunakan camel system.

Apa itu camel system?

Seperti kita ketahui, camel atau unta mempunyai punuk yang membentuk kurva naik turun. Nah dalam konteks bahasa pemrograman, camel system adalah cara penulisan kata dengan menggunakan huruf kecil dan membuat setiap kata diawali dengan huruf besar. Dengan penulisan seperti ini, maka rangkaian kata-kata tersebut akan lebih mudah dibaca.

Contoh camel system: IniAdalahCamelSystem

Jika kita mengikuti aturan penamaan variable yang diawali dengan huruf kecil, maka huruf i pada kata ini haruslah huruf kecil karena nama variable diawali dengan huruf kecil, sehingga nama variable kita menjadi: iniAdalahCamelSystem.

Perlu diingat, bahwa di banyak bahasa pemrograman termasuk Java, nama variable itu case-sensitive. Case-sensitive berarti bahwa huruf besar dan huruf kecil dianggap berbeda. Oleh karena itu, var1 dengan huruf v kecil dan Var1 dengan huruf V besar adalah dua variable yang berbeda. Selain itu, kita tidak diperbolehkan untuk mendeklarasikan variable dengan nama yang sama lebih dari satu kali. Contohnya : 

 int var1=5;
 int var1=8;  //integer var1 sama dengan 5, dan integer var1 sama dengan 8

Kode program di atas akan menghasilkan kesalahan atau error.

Tipe Data

Tipe data dari suatu variable menentukan jenis atau tipe dari nilai yang bisa kita simpan atau assign ke suatu variable. Jadi pada contoh di atas, variable var1 hanya bisa kita assign nilai yang bertipe int atau bilangan bulat. Nah mungkin akan muncul pertanyaan, apakah kita hanya bisa membuat variable dengan tipe bilangan bulat? Jawabannya adalah TIDAK.

Dalam bahasa pemrograman, terdapat beberapa tipe data yang umum bisa kita gunakan dan dapat dikelompokan menjadi 3 kategori, yaitu:

  • Bilangan
    Bilangan bisa berupa bilangan bulat ataupun desimal.
  • Karakter dan String
    Karakter (bahasa Inggris Character) adalah suatu simbol yang kita gunakan dalam bahasa tertulis dan dapat berupa huruf, angka, tanda baca, dolar, dll. Sedangkan string adalah deretan dari karakter yang bisa membentuk kata, kalimat ataupun sekedar deretan simbol yang tidak mempunyai arti.
  • Boolean
    Boolean adalah sebuah tipe data yang hanya mempunyai 2 kemungkinan nilai yaitu true (benar) atau false (salah).

Di Java, bilangan, karakter dan boolean termasuk tipe data sederhana atau primitif yang hanya bisa menyimpan satu nilai saja, sedangkan string diperlakukan sebagai object. Object termasuk tipe data kompleks yang bisa menyimpan banyak nilai. Kita akan membahas konsep object ini pada topik selanjutnya, oleh karena itu, ikuti terus tutorial kami.

Selain menjadikan bilangan sebagai tipe primitif, Java membagi lagi bilangan menjadi beberapa tipe. Berikut ini tipe data primitif yang ada di Java:

  • int
    Int adalah bilangan bulat yang nilainya antara -231 (minus dua pangkat tiga puluh satu ) hingga 231-1 (dua pangkat tiga puluh satu minus satu)
    Contoh deklarasi variable dengan tipe int:

     int var1 = 12345;  // integer var1 sama dengan satu dua tiga empat lima

  • byte
    Byte adalah bilangan bulat yang nilainya antara -128 (minus seratus dua puluh delapan) hingga 127 (seratus dua puluh tujuh)

    Contoh deklarasi variable dengan tipe byte:

     byte var2 = 1;  //byte var2 sama dengan satu
  • short
    Short adalah bilangan bulat yang nilainya antara -32768 (minus tiga puluh dua ribu tujuh ratus enam puluh delapan) hingga 32767 ( tiga puluh dua ribu tujuh ratus enam puluh tujuh)
    Contoh deklarasi variable dengan tipe short: 
     short var3 = -100;  //short var3 sama dengan minus seratus
  • long
    Long adalah bilangan bulat yang nilainya antara -263 (minus dua pangkat enam puluh tiga) hingga 263-1 (dua pangkat enam puluh tiga minus 1)
    Contoh deklarasi variable dengan tipe long: 
     long var4 = 12345678;  //long var4 sama dengan 1, 2, 3, 4, 5, 6, 7, 8
  • float
    Float adalah bilangan decimal 32-bit (bit adalah singkatan dari binary digit atau bilangan biner, merupakan bilangan yang hanya terdiri dari deretan angka 1 dan 0. 32-bit berarti deretan angka 1 dan 0 tersebut maximum 32 digit).
    Contoh deklarasi variable dengan tipe float: 
     float var5 = 1.2345;  //float var5 sama dengan 1 titik 2, 3, 4, 5
  • double
    Double adalah bilangan desimal 64-bit, yang bisa memuat nilai lebih besar dibandingkan float.
    Contoh deklarasi variable dengan tipe double: 
     double var6 = 12345.6789;  //double var6 sama dengan 12345 ttik 6789
  • char
    Char adalah karakter yang dituliskan di dalam petik tunggal
    Contoh deklarasi variable dengan tipe char: 
     char var7 = ‘a’;  //char var7 sama dengan dalam petik tunggal a
  • boolean
    Boolean adalah tipe variable yang hanya mempunyai 2 nilai yaitu true (benar) atau false (salah).
    Contoh deklarasi variable dengan tipe boolean: 
      boolean var8 = true;  //Boolean var8 sama dengan true
   boolean var9 = false;  //Boolean var9 sama dengan false

Selain tipe data sederhana, terdapat juga tipe data kompleks yang bisa menyimpan banyak nilai di dalam sebuah variable seperti array dan object. Kita akan membahas array dan object di topic selanjutnya. Oleh karena itu, ikutilah terus tutorial kami.
